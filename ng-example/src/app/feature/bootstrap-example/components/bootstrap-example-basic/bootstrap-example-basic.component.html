  • Automatic installation may encounter problems
  • Manual installation may be a better option
    1. Add bootstrap and @ng-bootstrap/ng-bootstrap dependencies from npm
    2. Install the @angular/localize polyfill (It is a dependencie of @ng-bootstrap/ng-bootstrap) - https://angular.io/guide/i18n#setting-up-localization-with-the-angular-cli
      "ng add @angular/localize" - It npm install the @angular/localize and add an entry to the polyfill.ts file.
    3. Add bootstrap CSS/SCSS to your project (no javascript is required)
    4. Import NgbModule or any other component module like NgbPaginationModule in your application
If we want to use only the css, we can just add the css "@import "~bootstrap/scss/bootstrap";" in the styles.scss file
